甄士隐梦幻识通灵 贾雨村风尘怀闺秀

>[!textus]
>此开卷第一回也。作者自云：因曾历过一番梦幻之后，故将真事隐去，而借“通灵”之说，撰此《石头记》一书也。故曰“甄士隐”云云。但书中所记何事何人？自又云：“今风尘碌碌，一事无成，忽念及当日所有之女子，一一细考较去，觉其行止见识皆出于我之上。何我堂堂须眉[^1]，诚不若彼裙钗[^2]哉？实愧则有馀，悔又无益之大无可如何之日也！当此，则自欲将已往所赖天恩祖德，锦衣<ruby>纨<rt>wán</rt></ruby><ruby>袴<rt>kù</rt></ruby>[^3]之时，<ruby>饫<rt>yù</rt></ruby>甘<ruby>餍<rt>yàn</rt></ruby>肥[^4]之日，背父兄教育之恩，负师友规训之德，以至今日一技无成，半生潦倒之罪，编述一集，以告天下人：**我之罪固不免，然闺阁中本自历历有人，万不可因我之不肖，自护己短，一并使其泯灭也。**<sub>*（这是一本为女子立传的书，因为不想让女子们的才华泯灭）*</sub>虽今日之茅<ruby>椽<rt>chuán</rt></ruby>蓬<ruby>牖<rt>yǒu</rt></ruby>[^5]，瓦灶绳床[^6]，其晨夕风露，阶柳庭花，亦未有妨我之襟怀笔墨者。虽我未学，下笔无文，又何妨用假语村言，敷演[^7]出一段故事来，亦可使闺阁昭传，复可悦世之目，破人愁闷，不亦宜乎？”故曰“贾雨村云云。<sub>*【* ***为何写?*** *经历人生幻灭后，深感有责任记录那些见识德行远超于己的优秀女性，不让她们的事迹因自己的无能而湮没。*  ***写什么?***  *闺阁中杰出女子的故事（“使闺阁昭传”）。*  ***怎么写?***   *将真事隐去（甄士隐），假托神话（通灵石头），用通俗虚构的语言（贾雨村/假语存）来演绎铺陈。* ***态度如何?*** *带着忏悔和使命感，虽身处贫困，但决心坚定，要用自己的方式完成这部作品，以期流传闺阁事迹并娱人解闷*  *】*</sub>
>```
>这本书开卷第一回，作者自己说：因为曾经经历过一番梦幻之后，所以把真实的事情隐去，而借助通灵的说法，撰写了《石头记》这本书。所以说“甄士隐梦幻识通灵”。但是书中所记载的是什么事情呢，又是因为什么而撰写了这本书呢？作者又说：如今在尘世中奔波忙碌，一事无成，忽然回想起往日所结识的所有女子，一一仔细推究起来，觉得她们的行为举止和见识，都在我之上。为什么我堂堂男子汉，竟然比不上那些女子呢！实在是羞愧得很，后悔又没有用处，这真是无可奈何的时候啊！在这种情况下，自己就想把以往所依赖的天恩祖德，过着穿绫罗绸缎、吃美味佳肴生活的时候，违背父兄的教育之恩，辜负师友的规劝之德，以至于到如今一事无成、半生穷困潦倒的罪过，编纂成册，昭告天下苍生。我的罪过固然不能免除，然而闺阁之中本来就有很多才华出众的女子，万万不可因为我不成器，为自己掩饰过错，而让她们的事迹也一同被埋没了。虽然如今居住着用茅草做椽子、蓬草做窗户的简陋屋子，使用着瓦做的炉灶、绳做的床，但那些晨风夕月、阶前垂柳、庭院鲜花，也不会影响我抒发胸怀和笔墨创作。虽然我没什么学问，写文章也没什么文采，但为何不能用些虚构的话语、村野的言辞，铺陈演绎出一段故事，来让世人赏心悦目呢？
>```

>原来女娲氏炼石补天[^9]之时，于大荒山无稽崖[^10]炼成高经十二丈，方经二十四丈顽石三万六千五百零一块。娲皇氏只用了三万六千五百块，只单单剩了一块未用，便弃在此山青埂峰下。谁知此石自经煅炼之后，灵性已通，因见众石俱得补天，独自己无材不堪入选，遂自怨自叹，日夜悲号惭愧。
>```
>原来，当年女娲氏炼石补天的时候，在大荒山无稽崖炼就了高十二丈、边长二十四丈的顽石三万六千五百零一块。女娲氏只用了三万六千五百块石头，单单剩下了一块没有用，就把它丢弃在这座山的青埂峰下。谁知道这块石头自从经过煅烧熔炼之后，已经通了灵性，因为看到其他石头都能去补天，唯独自己没有才能，不能被选用，于是自怨自艾，日夜悲叹惭愧。
>```

> 【四大名著的传奇，皆由四块石头铺就： 《红楼梦》是顽石堕入红尘，亲历情天孽海、见证家族兴衰的时代悲歌；《三国演义》是群雄逐鹿，围绕传国玉玺展开的权力博弈与智谋较量；《水浒传》乃镇魔碑下，边缘豪杰啸聚山林，谱写一曲底层义气文化的血色挽歌；而《西游记》，则讲述一块仙石从懵懂无知到历经体制规训、文化洗礼，终入金箍的“驯化”之路】

>[!textus]
>一日，正当嗟悼之际，俄见一僧一道远远而来，生得骨格不凡，丰神迥异，说说笑笑来至峰下，坐于石边高谈快论。先是说些云山雾海神仙玄幻之事，后便说到红尘中荣华富贵；此石听了，不觉打动凡心，也想要到人间去享一享这荣华富贵，但自恨粗蠢，不得已，便口吐人言，向那僧道说道：“大师，弟子蠢物，不能见礼了。适闻二位谈那人世间荣耀繁华，心切慕之。弟子质虽粗蠢，性却稍通；况见二师仙形道体，定非凡品，必有补天济世之材，利物济人之德。如蒙发一点慈心，携带弟子得入红尘，在那富贵场中，温柔乡里受享几年，自当永佩洪恩，万劫不忘也。”二仙师听毕，齐憨笑道：“善哉，善哉！**那红尘中有却有些乐事，但不能永远依恃，况又有‘美中不足，好事多魔’八个字紧相连属，瞬息间则又乐极悲生，人非物换，究竟是到头一梦，万境归空，倒不如不去的好。** <sub>*（不经历一番，如何看得透？人生一世，不过一场体验。激流勇进是一种，暂避锋芒也是一种；纵情山水是一种，安于平淡亦是一种……无需执念太重，想尝哪种滋味，便去尝试。人生贵在走过，而非结果）*</sub>
>```
>一天，正当这块石头悲叹哀伤的时候，忽然看见一僧一道远远走来，两人仙风道骨，超凡脱俗，说说笑笑来到青埂峰下，坐在石头边高谈阔论。开始时谈论些云山雾海的神仙传说和玄幻故事，后来便说到了人世间的荣华富贵。这块石头听了，不觉动了凡心，也想到人间去享一享荣华富贵，但自恨粗蠢，不得已，便口吐人言，向那僧道说道：“大师！弟子是个蠢笨的东西，不能行礼了。刚才听二位谈论人世间的荣耀繁华，我心里十分羡慕向往。弟子虽然资质粗笨，但也有些悟性，况且看到二位大师仙风道骨，定然不是凡人，必定具有补救天阙、济助世人的才能，具备利益万物、救济众生的德行。如果承蒙二位发一点慈悲之心，携带弟子进入尘世，在那富贵场中、温柔乡里享受几年，我自然会永远铭记二位的大恩，历经万劫也不会忘记。两位仙师听完，一齐憨笑着说：“善哉，善哉！”那红尘中确实有些乐事，但不能永远依赖。何况又有‘美中不足，好事多魔’八个字紧密相连。转眼间就会乐到极点而悲伤降临，人已不同，景物变换。说到底最终就像一场梦，所有的景象都将归于虚无。倒不如不去的好。
>```

>[!textus]
>这石凡心已炽，那里听得进这话去，乃复苦求再四。二仙知不可强制，乃叹道：“此亦静极思动，无中生有之数也。既如此，我们便携你去受享受享，只是到不得意时，切莫后悔。”石道：“自然，自然。”那僧又道：“若说你性灵，却又如此质蠢，并更无奇贵之处。如此也只好踮脚[^11]而已。也罢，我如今大施佛法助你助，待劫终之日，复还本质，以了此案。你道好否？”石头听了，感谢不尽。**那僧便念咒书符，大展幻术，将一块大石登时变成[^12]一块鲜明莹洁的美玉，且又缩成扇坠[^13]大小的可佩可拿。那僧托于掌上，笑道：“形体倒也是个宝物了！** <sub>*（从真顽石到假宝玉，一块美玉的本质是顽石，启示我们：华美皮囊或藏空洞灵魂；堆砌的财富亦可能是幻化的泥污；炽热的情话常是未经淬炼的幼稚；对网红美食的趋之若鹜，其本质犹如屎壳郎逐粪；而世间林林总总的观念，大多只是盲从的惯性……凡有所相，皆是虚妄。若能穿透表象、识破非相，即近真相）*</sub>还只没有实在的好处，须得再镌上数字，使人一见便知是奇物方妙。然后携你到那<u class="dashed">昌明隆盛之邦，诗礼<ruby>簪<rt>zān</rt></ruby>缨之族[^14]，花柳繁华地，温柔富贵乡</u><sub>*（“昌明隆盛之邦”指一个兴旺强盛的朝代（伏笔后文“长安大都”）； “诗礼簪缨之族”指世代读书明礼、官宦显贵的家族（“诗礼”喻文化传承，“簪缨”代指高官厚禄，伏笔“荣国府”）； “花柳繁华地，温柔富贵乡”形容一个花柳繁盛、物阜民丰的繁华富贵之地（伏笔“大观园”与“紫芸轩”））*</sub>去安身乐业。”石头听了，喜不能禁，乃问：“不知赐了弟子那几件奇处，又不知携了弟子到何地方？望乞明示，使弟子不惑。”那僧笑道：“你且莫问，日后自然明白的。”说着，便袖了这石，同那道人飘然而去，竟不知投奔何方何舍。
>```
>这块石头凡心已经十分炽热，哪里听得进这番话，于是又反复苦苦哀求。两位仙人知道不能强行制止它，于是叹息道：“这也是静极思动、无中生有的定数啊。”既然如此，我们就带你去享享福，只是到了不如意的时候，千万不要后悔。石头说：“当然，当然。”那僧人又说：“若说你有灵性，却又如此质地蠢笨，并且毫无奇特珍贵之处，这样也只能勉强当个垫脚石罢了。”也罢，我如今大力施展佛法帮你一下，等劫数结束的那天，你再恢复本来面目，了结这段公案。你觉得这样好吗？石头听了，感激不尽。那和尚便念动咒语、书写符箓，充分施展幻术，把一块大石头瞬间变成了一块明亮洁净的美玉，并且又缩小成扇坠大小，成为可以佩戴、拿在手中的饰物。那僧人把它托在手掌上，笑着说：“形体倒也是个宝物了，只是还没有实际的功用，得再刻上几个字，让人一看见就知道是奇异的东西才好。”然后好带你到那政治清明、繁荣昌盛的国家，讲究诗礼、世代做官的家族，繁花似锦、繁华热闹的地方，温柔多情、富贵安逸的环境中去安身立业。石头听了，高兴得不能控制自己，便问道：“不知赐给了弟子哪几件奇特之处，又不知要带弟子到什么地方去？希望您明确指示，让我不再疑惑。那和尚笑着说：“你暂且别问，日后自然会明白的。”说着，便将那块石头放入袖中，和那道人一起飘然而去，竟不知他们去了什么地方。
>```

>[!textus]
后来，又不知过了几世几劫[^15]，因有个空空道人访道求仙，忽从这大荒山无稽崖青埂峰下经过，忽见一大块石上字迹分明，编述历历。空空道人乃从头一看，原来就是无材补天，幻形入世，蒙茫茫大士渺渺真人携入红尘，历尽离合悲欢炎凉世态的一段故事。后面又有一首<ruby>偈<rt>jì</rt></ruby>[^16]云：
> ```
> 后来，也不知道又过了几世几劫。有个空空道人四处访道求仙，偶然从这大荒山无稽崖青埂峰下经过，忽然看见一大块石头上字迹清楚，所编所述十分清晰。后面又有一首偈语说：
> ```

>[!textus]
>无材可去补苍天，枉入红尘若许年。
> ```
> 没有才能去修补苍天，白白地在这尘世间经历了这么多年！
> ```

>[!textus]
>此系身前身后事，倩谁[^17]记去作奇传？诗后便是此石坠落之乡，投胎之处，亲自经历的一段陈迹故事。其中家庭闺阁琐事，以及闲情诗词倒还全备，或可适趣解闷，然朝代年纪、地舆邦国，却反失落无考。
> ```
> 这讲述的是石头下凡前后的经历。请谁寄去当作奇闻流传呢？诗的后面便是这块石头堕落的地方、投胎的处所，以及它亲自经历的一段过去的事迹故事。其中记载的家庭闺阁中的琐事，以及闲暇时所作的诗文倒还十分齐全，或许可以用来舒缓心情、排解烦闷，然而朝代纪年、地理邦国，却反而失落无考了。
> ```

>[!textus]
>空空道人遂向石头说道：“石兄，你这一段故事，据你自己说有些趣味，故编写在此，意欲问世传奇。据我看来，第一件，无朝代年纪可考；第二件，并无大贤大忠理朝廷治风俗的善政，其中只不过几个异样女子，或情或痴，或小才微善，亦无班姑蔡女之德能[^18]。我纵抄去，恐世人不爱看呢。”石头笑答道：“我师何太痴耶！若云无朝代可考，今我师竟假借汉唐等年纪添缀，又有何难？**但我想，历来野史[^19]，皆蹈一辙，莫如我这不借此套者，反倒新奇别致，不过只取其事体情理罢了，又何必拘拘于朝代年纪哉！** <sub>*（“如果对现实感到不解，就读历史，因为一切早已发生；如果对历史感到困惑不解，就看现实，因为历史正在发生。”）* </sub>再者，市井俗人喜看理治之书[^20]者甚少，爱适趣闲文者特多。历来野史，或讪谤君相，或贬人妻女，奸淫凶恶，不可胜数[^21]。更有一种风月笔墨，其淫秽污臭，屠毒笔墨，坏人子弟，又不可胜数。至若佳人才子等书，则又千部共出一套，且其中终不能不涉于淫滥，以致满纸潘安、子建、西子、文君、不过作者要写出自己的那两首情诗艳赋来，故假拟出男女二人名姓，又必旁出一小人其间拨乱，亦如剧中之小丑然。且鬟婢开口即者也之乎，非文即理。故逐一看去，悉皆自相矛盾，大不近情理之话，竟不如我半世亲睹亲闻的这几个女子，虽不敢说强似前代书中所有之人，但事迹原委，亦可以消愁破闷；也有几首歪诗熟话，可以喷饭供酒。至若离合悲欢，兴衰际遇，则又追踪蹑迹，不敢稍加穿凿，徒为供人之目而反失其真传者。今之人，贫者日为衣食所累，富者又怀不足之心，纵然一时稍闲，又有贪淫恋色，好货寻愁之事，那里去有工夫看那理治之书？所以我这一段故事，也不愿世人称奇道妙，也不定要世人喜悦检读，只愿他们当那醉淫饱卧[^22]之时，或避事[^23]去愁之际，把此一玩，岂不省了些寿命筋力？就比那谋虚逐妄，却也省了口舌是非之害，腿脚奔忙之苦。再者，亦令世人换新眼目，不比那些胡牵乱扯，忽离忽遇，满纸才人淑女、子建文君红娘小玉[^24]等通共熟套之旧稿。我师意为何如？”
> ```
> 空空道人于是对石头说：“石兄，你这段故事，据你自己说有些趣味，所以编写在这里，想要在世上流传成为传奇。依我看来，第一件，没有可以考证的朝代年纪；第二件，没有大贤大忠之人治理朝廷、整顿风俗的善政，其中只不过几个与众不同的女子，有的多情，有的痴迷，有一点小才华和微小的善举，也没有班姑、蔡女那样的德行才能。我就算把它抄录下来，恐怕世人也不会喜欢看。石头笑着回答说：“我的师父怎么这么痴傻啊！如果说没有朝代可以考证，如今您竟要假借汉朝、唐朝等朝代的纪年进行添补，这又有什么难的呢？但我想，历来的野史，都走一样的套路，不如我这不借用这种套路的，反倒新奇别致，不过只是选取其中的事情道理罢了，又何必拘泥于朝代年纪呢！再者，市井中的普通百姓喜欢看讲述治理国家道理书籍的人很少，喜欢看有趣味的闲杂文章的人特别多。历来的野史，有的讥讽毁谤君主和宰相，有的贬低他人的妻子女儿，其中描述的奸淫凶恶之事，多得数都数不过来。还有一种描写男女情爱的文章，它们内容淫秽不堪，文字如同毒药般害人，毒害年轻人，这样的作品多得数都数不清。至于那些描写佳人才子的书，更是成千上万部都一个套路，而且书中终究免不了涉及淫秽放荡的内容，以至于满纸都是潘安、曹植、西施、卓文君这样的人物，不过是作者想要写出自己的那两首情诗艳赋，所以假想出男女两个人的姓名，还一定会另外安排一个小人在中间挑拨捣乱，就像戏剧里的小丑一样。而且丫鬟一开口就是之乎者也，不是谈文就是论理。所以逐一看来，都是自相矛盾、极不合情理的话。反倒不如我半生亲眼所见、亲耳所闻的这几个女子，虽说不敢说比前代书中所写的人都强，但她们的事迹始末，也能让人消愁解闷，还有几首打油诗和俏皮话，可以让人发笑，供饮酒时取乐。至于书中人物的离合悲欢、兴衰际遇，都是按照实际情况去描写，不敢有丝毫的虚构和穿凿附会，生怕只是为了博人眼球，反而失去了真实的韵味。如今的人，贫穷的每天被衣食所累，富贵的又怀着不满足的贪心。即便一时稍微有空，又都沉迷于贪淫好色、追逐钱财、自寻烦恼的勾当，哪里有工夫去看那些讲治理之道的书呢？所以我这一段故事，既不希望世人称赞它奇妙，也不要求世人高兴地阅读，只希望他们在醉酒纵欲或饱食昏睡时，或者躲避世事、排解忧愁的时候，把它拿来看一看，这样岂不是节省了些寿命和精力？比起那些追求虚幻不实之事，这样反而避免了口舌是非的祸害，省去了腿脚奔波忙碌的辛苦。再者，也能让世人换个新的视角，不像那些胡编乱造、情节忽离忽遇，满纸都是才子佳人、子建文君、红娘小玉等千篇一律旧套路的稿子。老师您觉得怎么样？
> ```

>[!textus]
>空空道人听如此说，思忖半晌，将《石头记》再检阅一遍，因见上面虽有些指奸责佞贬恶诛邪之语，亦非伤时骂世之旨；及至君仁臣良父慈子孝，凡伦常[^25]所关之处，皆是称功颂德，眷眷无穷，实非别书之可比。虽其中大旨谈情，亦不过实录其事，又非假拟妄称，一味淫邀艳约、私订偷盟之可比。因毫不干涉时世，方从头至尾抄录回来，问世传奇。从此空空道人[^26]因空[^27]见色，由色生情，传情入色，自色悟空，遂易名为情僧，改《石头记》为《情僧录》。东鲁孔梅溪则题曰《风月宝鉴》[^28]。后因曹雪芹于悼红轩中披阅十载，增删五次，纂成目录，分出章回，则题曰《金陵十二钗》[^29]。并题一绝云：
> 
> 满纸荒唐言，一把辛酸泪！
> 都云作者痴，谁解其中味？
> 
> ```
> 空空道人听了这番话，思索了半晌，将《石头记》又重新检查阅读了一遍，发现上面虽然有一些揭露奸恶、斥责谗佞、贬斥邪祟的话语，但并非是那种感伤时世、咒骂社会的主旨，至于写到君主仁爱、臣子贤良、父亲慈爱、子女孝顺等凡是与伦理纲常有关的地方，都是在称赞功德、颂扬美德，情意恳切，实在不是其他书可以相比的。虽然书中的主旨是谈情说爱，但也不过是如实记录事情，并非虚构妄加称道，与那种一味描写男女淫荡地邀约、私下订立盟约的情况不可相提并论。因为完全不涉及干涉当时的社会世事，这才从头到尾抄录回来，让它在世间流传成为传奇。因为从空寂的境界见到了色相，由色相产生了情感，将情感传递到色相之中，又从色相里领悟到空寂的道理，于是改名为情僧，把《石头记》改名为《情僧录》。到了吴玉峰，将其题名为《红楼梦》。东鲁的孔梅溪将其题名为《风月宝鉴》。后来因为曹雪芹在悼红轩里批阅了十年，增删了五次，编成目录，分出章回，就题名为《金陵十二钗》。并且题写了一首绝句说：
> 
> 满纸都是荒唐的言辞，饱含着一把辛酸的眼泪！
> 都说作者痴傻，谁能理解其中的滋味呢？
> ```

>[!textus]
>当日地陷东南[^30]，这东南一隅有处曰姑苏，有城曰<ruby>阊<rt>chāng</rt></ruby>门[^31]者，最是红尘中一二等富贵风流之地。这阊门外有个十里街，街内有个仁清巷[^32]，巷内有个古庙，因地方窄狭，人皆呼作葫芦庙。庙旁住着一家乡宦，姓甄，名费，字士隐。嫡妻封氏，情性贤淑，深明礼义。家中虽不甚富贵，然本地便也推他为望族了。因这甄士隐禀性恬淡，不以功名为念，每日只以观花修竹、酌酒吟诗为乐，倒是神仙一流人品。只是一件不足：如今年已半百，膝下无儿，只有一女，乳名唤作英莲[^33]，年方三岁。

> 当时大地向东南塌陷，这东南一角有个地方叫姑苏，有座城叫阊门，它是人世间一二等富贵风流的好地方。这阊门外有一条十里街，街内有一条仁清巷，巷内有一座古庙，因为地方狭窄，人们都叫它葫芦庙。庙旁住着一家做过官的人家，姓甄，名费，字士隐。正妻封氏，性情贤淑，深明礼义。家中虽然不是非常富贵，但在当地也被公认为是有声望的家族了。因为这甄士隐生性淡泊宁静，不把功名利禄放在心上，每天只以观赏花卉、修剪竹子、饮酒赋诗为乐，倒真是有神仙一样的品性。只是有一件事美中不足：如今他已年过半百，膝下没有儿子，只有一个女儿，乳名叫英莲，年仅三岁。

>[!textus]

> ```

>[!textus]
>只听道人问道：“你携了这蠢物，意欲何往？”那僧笑道：“你放心，如今现有一段风流公案正该了结，这一干风流冤家[^35]，尚未投胎入世。趁此机会，就将此蠢物夹带于中，使他去经历经历。”那道人道：“原来近日风流冤孽又将造劫历世去不成？但不知落于何方何处？”那僧笑道：“此事说来好笑，竟是千古未闻的罕事。只因西方灵河岸上三生石[^36]畔，有绛珠草一株，时有赤瑕宫神瑛侍者，日以甘露灌溉，这绛珠草始得久延岁月。后来既受天地精华，复得雨露滋养，遂得脱却草胎木质，得换人形，仅修成个女体，终日游于离恨天外，饥则食蜜青果为膳，渴则饮灌愁海[^37]水为汤。只因尚未酬报灌溉之德，故其五内[^38]便郁结着一段缠绵不尽之意。恰近日这神瑛侍者凡心偶炽，乘此昌明太平朝世，意欲下凡造历幻缘，已在警幻仙子案前挂了号。警幻亦曾问及，灌溉之情未偿，趁此倒可了结的。那绛珠仙子道：‘他是甘露之惠，我并无此水可还。他既下世为人，我也去下世为人，但把我一生所有的眼泪还他，也偿还得过他了。’因此一事，就勾出多少风流冤家来，陪他们去了结此案。”
> ```
> 只听道人问道：“你带着这个蠢物，打算去哪里？”那僧笑道：“你放心，如今正好有一段风流公案该了结，这一群风流冤家，还没投胎到人世。”趁着这个机会，就把这个蠢物夹带在里面，让他去经历一番。那道人说：“原来近日这些风流冤孽又要去人间遭受劫难、经历尘世了吗？但不知会落到何方何地呢？”那僧人笑着说：“这件事说起来好笑，竟然是千古以来都没听说过的稀罕事。”只因为西方灵河岸上三生石旁边，有一株绛珠草，当时有赤瑕宫的神瑛侍者，每天用甘露浇灌它，这株绛珠草才能长久存活。后来这绛珠草吸收了天地的精华，又得到雨露的滋养，于是脱离了草木的形态，化为人形，仅仅修炼成了女子的身躯，整日在离恨天之外游荡，饿了就吃蜜青果充饥，渴了就喝灌愁海水解渴。只因还没有报答灌溉的恩德，所以她内心深处便郁结着一段缠绵不尽的情意。恰好近日这神瑛侍者突然动了凡心，趁着如今这昌盛清明、太平的时代，想要下凡去经历一段虚幻的缘分，已经在警幻仙子那里挂了号。警幻仙子也曾经问起绛珠草还未报答灌溉之恩的事情，趁着这次机会，倒是可以了结了。那绛珠仙子说：“他给予我甘露的恩惠，我并没有这样的水可以偿还。他既然要下凡为人，我也下凡为人，把我一生所有的眼泪还给他，也就偿还得过他了。因为这一件事，就引出了多少风流冤家，陪着他们去了结这桩公案。”

>士隐接了看时，原来是块鲜明美玉，上面字迹分明，镌着“通灵宝玉”四字，后面还有几行小字。正欲细看时，那僧便说已到幻境，便强从手中夺了去，与道人竟过一大石牌坊，上书四个大字，乃是“太虚幻境[^44]”。两边又有一幅对联，道是：
> 
> 假作真时真亦假，无为有处有还无[^45]。
> 
> ```
> 士隐接过来一看，原来是一块晶莹剔透的美玉，上面的字迹清晰可见，镌刻着“通灵宝玉”四个字，后面还有几行小字。正想仔细看时，那僧人便说已经到了幻境，就强行从他手中夺过了玉，与道人径直穿过一座大石牌坊，牌坊上刻着四个大字，正是‘太虚幻境’。
> 
> 把假的当作真的时，真的也就成了假的；把没有的当作有的，有的也就成了没有的。
> ```

>[!textus]
>士隐意欲也跟了过去，方举步时，忽听一声霹雳，有若山崩地陷。士隐大叫一声，定睛一看，只见烈日炎炎，芭蕉冉冉，所梦之事便忘了大半。又见奶母正抱了英莲走来。士隐见女儿越发生得粉妆玉琢，乖觉可喜，便伸手接来，抱在怀内，斗他顽耍一回，又带至街前，看那过会[^46]的热闹。
> ```
> 甄士隐打算也跟着过去，刚要迈步时，忽然听到一声霹雳，好像山崩地陷一般。士隐大叫一声，定睛一看，只见烈日炽热，芭蕉轻轻摇曳，所梦之事便忘了大半。士隐又看见奶母正抱着英莲走过来。士隐见女儿越发长得粉雕玉琢，乖巧伶俐惹人喜爱，便伸手接过来，抱在怀里，逗她玩耍了一会儿，又带她到街前，看那过会的热闹场面。
> ```

>[!textus]
>方欲进来时，只见从那边来了一僧一道：那僧则癞头跣脚，那道则跛足蓬头，疯疯癫癫，挥霍[^47]谈笑而至。及至到了他门前，看见士隐抱着英莲，那僧便大哭起来，又向士隐道：“施主，你把这有命无运[^48]、累及爹娘之物，抱在怀内作甚？”士隐听了，知是疯话，也不去睬他。那僧还说：“舍我罢，舍我罢！”士隐不耐烦，便抱女儿撤身要进去，那僧乃指着他大笑，口内念了四句言词道：
>
惯养娇生笑你痴，菱花空对雪澌澌[^49]。
好防佳节元宵后，便是烟消火灭时。
> ```
> 正要进来的时候，只见从那边走来一僧一道，那和尚满头癞痢、光着脚，那道士瘸着腿、头发蓬乱，疯疯癫癫、洒脱地谈笑走来。等到来到士隐门前，看见士隐抱着英莲，那和尚便大哭起来，又对士隐说：“施主，你把这个有命无运、连累爹娘的东西抱在怀里做什么？”士隐听了，知道他说的是疯话，便不去理睬他。那僧人还说：“把她给我吧，把她给我吧！”士隐不耐烦，便抱着女儿转身要进去，那僧人指着他大笑，嘴里念了四句诗道：
> 
> 可笑你痴傻地娇惯养育孩子，菱花白白地对着那纷纷扬扬的大雪。
> 千万要提防元宵佳节之后，就是你家一切都烟消云散的时候！
> ```


>[!textus]
>士隐听得明白，心下犹豫，意欲问他们来历。只听道人说道：“你我不必同行，就此分手，各干营生去罢。三劫后，我在北<ruby>邙<rt>mánɡ</rt></ruby>山[^50]等你，会齐了同往太虚幻境销号。”那僧道：“最妙，最妙！”说毕，二人一去，再不见个踪影了。士隐心中此时自忖：这两个人必有来历，该试一问，如今悔却晚也。
> ```
> 士隐听得真切，心里犹豫不决，想要问他们的来历。只听道人说道：“你我不必同行，就此分手，各自去做自己的事吧。三劫之后，我在北邙山等你，咱们集齐了一起前往太虚幻境销号。那和尚说：“妙啊，妙啊，妙啊！”说完之后，两人一离去，就再也看不见踪影了。士隐心中此时暗自思量：这两个人必定大有来历，应该试探着问一问，现在后悔却已经晚了。
> ```

>[!textus]
>这士隐正痴想，忽见隔壁葫芦庙内寄居的一个穷儒──姓贾名化、表字时飞、别号雨村者走了出来。这贾雨村原系胡州[^51][^52]人氏，也是诗书仕宦之族，因他生于末世，父母祖宗根基已尽，人口衰丧，只剩得他一身一口，在家乡无益，因进京求取功名，再整基业。自前岁来此，又淹<ruby>蹇<rt>jiǎn</rt></ruby>[^53]住了，暂寄庙中安身，每日卖字作文为生，故士隐常与他交接。
> ```
> 这甄士隐正在痴痴地想着，忽然看见隔壁葫芦庙内寄居的一个穷书生，这个人姓贾名化，表字时飞，别号雨村，走了出来。这贾雨村原本是胡州人，出身于书香官宦世家，因为他生在家族衰败的时代，父母祖宗的基业已经耗尽，家族人口凋零，只剩下他孤身一人，在家乡难以立足。于是进京去考取功名，重新振兴家族基业。自从前年来到这里，又因命运不佳而停滞下来，暂时寄居在庙中安身，每天靠卖字写文章维持生计，所以士隐常常与他来往。
> ```

>[!textus]
>当下雨村见了士隐，忙施礼陪笑道：“老先生倚门伫望，敢[^54]是街市上有甚新闻否？”士隐笑道：“非也。适因小女啼哭，引他出来作耍，正是无聊之甚，兄来得正妙，请入小斋一谈，彼此皆可消此永昼。”说着，便令人送女儿进去，自与雨村携手来至书房中。小童献茶。方谈得三五句话，忽家人飞报：“严老爷来拜。”士隐慌的忙起身谢罪道：“恕诳驾[^55]之罪，略坐，弟即来陪。”雨村忙起身亦让道：“老先生请便。晚生乃常造之客，稍候何妨。”说着，士隐已出前厅去了。
> ```
> 当下贾雨村见到了甄士隐，连忙行礼赔笑道：“老先生站在门口张望，莫不是街市上有什么新鲜事？”士隐笑着说：“不是的，刚才因为小女儿啼哭，带她出来玩耍，正觉得无聊得很，兄来得正妙，请进我的书房聊一聊，彼此都可以消磨这漫长的白天。”说着，便让人送女儿进去，自己与雨村携手来到书房中。小童献上茶来。刚刚谈了三五句话，忽然家人飞奔来报告：“严老爷来拜访。”士隐慌得连忙起身谢罪说：“请原谅我诓驾之罪，您稍坐片刻，我马上就来陪您。”雨村急忙站起来，谦让着说：“老先生请自便。”晚生是经常来拜访的客人，稍等片刻也没什么关系。说着，士隐已经到前厅去了。
> ```

>[!textus]
>这里雨村且翻弄书籍解闷。忽听得窗外有女子嗽声，雨村遂起身往窗外一看，原来是一个丫鬟，在那里<ruby>撷<rt>xié</rt></ruby>[^56]花，生得仪容不俗，眉目清明，虽无十分姿色，却亦有动人之处。雨村不觉看的呆了。
> ```
> 这时雨村暂且翻弄书籍来排解烦闷。忽然听到窗外有女子咳嗽的声音，雨村于是起身走到窗边向外张望，原来是一个丫鬟，正在那里摘花，生得容貌气质不俗，眉眼清秀，虽然没有十分出众的姿色，却也有动人的地方。雨村不知不觉看呆了。
> ```

>[!textus]
>那甄家丫鬟撷了花，方欲走时，猛抬头见窗内有人，敝巾旧服，虽是贫窘，然生得腰圆背厚，面阔口方，更兼剑眉星眼，直鼻权腮[^57]。这丫鬟忙转身回避，心下乃想：“这人生的这样雄壮，却又这样褴褛，想他定是我家主人常说的什么贾雨村了，每有意帮助周济，只是没甚机会。我家并无这样贫窘亲友，想定是此人无疑了。怪道又说他必非久困之人。”如此想来，不免又回头两次。雨村见他回了头，便自为这女子心中有意于他，便狂喜不尽，自为此女子必是个巨眼英雄[^58]，风尘[^59]中之知己也。一时小童进来，雨村打听得前面留饭，不可久待，遂从夹道中自便出门去了。士隐待客既散，知雨村自便，也不去再邀。
> ```
> 那甄家丫鬟摘了花，刚要走的时候，猛一抬头看见窗内有人，戴着破头巾、穿着旧衣服，虽然处境贫困窘迫，然而长得腰圆背厚，脸宽嘴方，更兼剑眉下星眸璀璨，直鼻梁、方脸颊。这个丫鬟急忙转身避开，心里想：“这人长得如此魁梧，却穿得这样破旧，想来他一定是我家主人常说起的那个贾雨村了，主人常常有意帮助周济他，只是一直没什么机会。”我家并没有这样贫穷窘迫的亲友，想来一定是这个人没错了。怪不得又说他一定不是长久处于困境的人。这样想着，不免又回头看了两次。雨村见她回了头，便认为这个女子对自己有意，于是欣喜若狂，自认为这个女子一定是个眼光独到、慧眼识珠的豪杰，是自己在尘世中的知己。这时，小童进来，雨村打听得知前面留客人吃饭，自己不便久留，于是从夹道中自行出门去了。甄士隐等客人散去后，知道贾雨村已经自行离开，便不再去邀请他了。
> ```

>[!textus]
>一日，早又中秋佳节。士隐家宴已毕，乃又另具一席于书房，却自己步月至庙中来邀雨村。原来雨村自那日见了甄家之婢曾回顾他两次，自为是个知己，便时刻放在心上。今又正值中秋，不免对月有怀，因而口占五言一律[^60]云：
> 
> 未卜[^61]三生愿，频添一段愁[^62]。
> 闷来时敛额[^63]，行去几回头。
> 自顾风前影，谁堪月下俦？[^64]
> 蟾光如有意，先上玉人楼。[^65]
> 
> ```
> 一天，很快又到了中秋佳节。甄士隐家宴结束后，又在书房摆了一桌酒席，自己则踏着月光到庙中去邀请贾雨村。原来雨村自从那天看见甄家的丫鬟回头看了他两次，就认为这个丫鬟是自己的知己，于是时刻把这件事放在心上。如今又正赶上中秋佳节，不免对着月亮有所感怀，于是随口吟出一首五言律诗道：
> 
> 无法预料三生的愿望能否实现，却频繁增添了一段忧愁。
> 烦闷的时候皱起眉头，离开时频频回头。
> 独自看着风中的影子，谁能成为我月下的伴侣呢？
> 月光若懂得我的心意，就先照到美人居住的楼阁。
> ```
   
>[!textus]
>雨村吟罢，因又思及平生抱负，苦未逢时，乃又搔首对天长叹，复高吟一联曰：
>
玉在椟中求善价，钗于奁内待时飞。[^66]
> ```
> 雨村吟诵完毕，又想到平生的志向抱负，苦于没有遇到好时机，于是挠头对着天空长叹，接着高声吟诵了一副对联说道：
> 
> 美玉放在匣子里等待识货的商人出高价来买，金钗放在妆盒里等待时机要飞向天际。
> ```

>[!textus]
>恰值士隐走来听见，笑道：“雨村兄真抱负不浅也！”雨村忙笑道：“不过偶吟前人之句，何敢狂诞至此。”因问：“老先生何兴至此？”士隐笑道：“今夜中秋，俗谓‘团圆之节’，想尊兄旅寄僧房，不无寂寥之感，故特具小酌，邀兄到敝斋一饮，不知可纳芹意[^67]否？”雨村听了，并不推辞，便笑道：“既蒙厚爱，何敢拂此盛情。”说着，便同士隐复过这边书院中来。
> ```
> 恰巧士隐走过来听见了，笑着说：“雨村兄的抱负真不浅啊！”雨村连忙笑着说：“不过是偶然吟诵前人的诗句，怎敢狂妄到这种地步。”于是问道：“老先生怎么有兴致到这里来了？”士隐笑道：‘今夜是中秋，俗话说这是‘团圆之节’，想来尊兄客居在僧房，难免会有寂寥之感，所以特地备了薄酒，邀请尊兄到我家小聚一杯，不知能否接受我的一番心意？’雨村听了，并不推辞，便笑着说：“既然承蒙您厚爱，怎敢拒绝这番盛情。”说着，便和士隐一起再次来到这边的书院里。
> ```



>[!textus]
>须臾茶毕，早已设下杯盘，那美酒佳肴自不必说。二人归坐，先是款斟漫饮，次渐谈至兴浓，不觉飞<ruby>觥<rt>ɡōnɡ</rt></ruby>限<ruby>斝<rt>jiǎ</rt></ruby>[^68]起来。当时街坊上家家箫管，户户弦歌，当头一轮明月，飞彩凝辉，二人愈添豪兴，酒到杯干。雨村此时已有七八分酒意，狂兴不禁，乃对月寓怀，口号[^69]一绝云：
时逢三五[^70]便团圆，满把晴光护玉栏。[^71]
天上一轮才捧出，人间万姓仰头看。[^72]
> ```
> 不一会儿喝完茶，早已摆好了杯盘，那美酒佳肴自然就不必说了。两人回到座位上，先是慢慢斟酒、细细品尝，接着渐渐谈到兴致浓厚，不知不觉就开始开怀畅饮起来。当时街坊上家家户户都传出箫管之声，弹奏着弦乐，头顶悬着一轮明月，光彩流动辉映四方，两人更添豪迈兴致，酒一倒进杯中就喝干。雨村这时已经有了七八分酒意，狂放的兴致难以抑制，于是对着月亮寄托情怀，随口吟出一首绝句说：。
> 
> 每到十五月亮就会团圆，满手的明朗月光守护着玉石栏杆。
> 天上的一轮明月刚刚升起。天下百姓都仰头瞻望。
> ```

>[!textus]
>士隐听了，大叫：“妙哉！吾每谓兄必非久居人下者，今所吟之句，飞腾之兆已见，不日可接履于云霓之上[^73]矣。可贺，可贺！”乃亲斟一斗为贺。雨村因干过，叹道：“非晚生酒后狂言，若论时尚之学[^74]，晚生也或可去充数沽名，只是目今行囊路费一概无措，神京[^75]路远，非赖卖字撰文即能到者。”士隐不待说完，便道：“兄何不早言。愚每有此心，但每遇兄时，兄并未谈及，愚故未敢唐突。今既及此，愚虽不才，‘义利’二字[^76]却还识得。且喜明岁正当大比，兄宜作速入都，春闱[^77]一战，方不负兄之所学也。其盘费馀事，弟自代为处置，亦不枉兄之谬识矣！”当下即命小童进去，速封五十两白银，并两套冬衣。又云：“十九日乃黄道之期，兄可即买舟西上，待雄飞高举，明冬再晤，岂非大快之事耶！”雨村收了银衣，不过略谢一语，并不介意，仍是吃酒谈笑。那天已交了三更，二人方散。
> ```
> 士隐听了，大声叫道：“妙啊！我常常说兄台必定不会长久屈居人下，如今所吟的诗句，已经显现出飞黄腾达的征兆，不久之后就可以平步青云了。真是值得庆贺，值得庆贺啊！于是亲自斟了一斗酒来祝贺。雨村喝干了酒，感叹道：“并非我酒后说大话，若是论当今流行的学问，我也许能去凑数博取名声，只是眼下行李、路费一概无着落，京城路途遥远，单靠卖字写文章是到不了那儿的。”士隐不等他说完，便说：“兄台为什么不早说。我一直有这个想法，但每次遇到您时，您都没有提及此事，所以我不敢贸然提起。如今既然谈到了此事，我虽然不才，‘义利’二字还是明白的。幸好明年正是科举大考之年，兄台应该尽快前往京城，参加春闱考试，这样才不会辜负兄台的所学。这盘缠费用以及其他的事，我自会替你安排处理，也不枉兄长对我的错爱了！当下立刻让小童进去，赶快封好五十两白银和两套冬衣。又说：“十九日是黄道吉日，兄台可立即雇船西上，等兄台金榜题名，明年冬天再相聚，难道不是一件大快人心的事吗！”雨村收下了银子和冬衣，只是简单地说了一句感谢的话，并不在意，仍旧一边喝酒一边谈笑。那天已经到了三更时分，两人才各自散去。
> ```

>[!textus]
>士隐送雨村去后，回房一觉，直至红日三竿方醒。因思昨夜之事，意欲再写两封荐书与雨村带至神都，使雨村投谒个仕宦之家为寄足之地。因使人过去请时，那家人去了回来说：“和尚说，贾爷今日五鼓已进京去了，也曾留下话与和尚转达老爷，说‘读书人不在黄道黑道[^78]，总以事理为要，不及面辞了。’”士隐听了，也只得罢了。
> ```
> 士隐送雨村走后，回到房里睡了一觉，直到太阳升得很高了才醒来。于是想到昨夜的事，打算再写两封推荐信让雨村带到神都，让雨村去拜访个官宦人家作为暂时的落脚之地。于是派人过去邀请，那家人去了回来报告说：‘和尚说，贾爷今天五更天已经进京去了，还曾留下话让和尚转告老爷，说：“读书人不必拘泥于黄道黑道，总归应以事理为重，来不及当面告辞了。”’。士隐听了，也只好就此作罢了。
> ```

>[!textus]
>真是闲处光阴易过，倏忽又是元宵佳节矣。士隐命家人霍启抱了英莲去看社火花灯[^79]，半夜中，霍启因要小解，便将英莲放在一家门槛上坐着。待他小解完了来抱时，那有英莲的踪影？急得霍启直寻了半夜，至天明不见，那霍启也就不敢回来见主人，便逃往他乡去了。那士隐夫妇，见女儿一夜不归，便知有些不妥，再使几人去寻找，回来皆云连音响皆无。夫妻二人，半世只生此女，一旦失落，岂不思想，因此昼夜啼哭，几乎不曾寻死。看看的一月，士隐先就得了一病，当时封氏孺人[^80]也因思女构疾，日日请医疗治。
> ```
> 真是在悠闲的日子里时间过得飞快，转眼间又到了元宵佳节了。士隐让家人霍启抱着英莲去看社火花灯，半夜里，霍启因为要小解，就把英莲放在一家门槛上坐着。等他小解完回来抱英莲的时候，哪里还有英莲的踪影？霍启急得一直寻找了半夜，到天亮也没找到，他不敢回去见主人，便逃往他乡去了。那士隐夫妇，见女儿一夜未归，便觉事情有些不妙，又派了几个人去寻找，回来都说连一点消息都没有。夫妻二人半辈子只生了这个女儿，一旦失去，怎么能不思念，因此日夜啼哭，几乎寻死。转眼一个月过去，士隐先就生了一场病，当时封氏夫人也因思念女儿患病，天天请医生来治疗。
> ```

>[!textus]
>不想这日三月十五，葫芦庙中炸供[^81]，那些和尚不加小心，致使油锅火逸，便烧着窗纸。此方人家多用竹篱木壁者，大抵也因劫数，于是接二连三，牵五挂四，将一条街烧得如火焰山一般。彼时虽有军民来救，那火已成了势，如何救得下？直烧了一夜，方渐渐的熄去，也不知烧了几家。只可怜甄家在隔壁，早已烧成一片瓦砾场了。只有他夫妇并几个家人的性命不曾伤了。急得士隐惟跌足长叹而已。只得与妻子商议，且到田庄上去安身。偏值近年水旱不收，鼠盗蜂起，无非抢田夺地，鼠窃狗偷，民不安生，因此官兵剿捕，难以安身。士隐只得将田庄都折变了，便携了妻子与两个丫鬟投他岳丈家去。
> ```
> 没想到这一天是三月十五，葫芦庙中炸供品，那些和尚不小心，致使油锅里的火窜了出来，烧着了窗纸。这地方的人家大多用竹篱笆和木板墙建造房屋，大概也是因为劫数，于是火势接二连三、牵连蔓延开来，将整条街烧得像火焰山一样。当时虽然有军民赶来救火，但火势已经蔓延开来，怎么能救得下来呢？大火一直烧了一夜，才渐渐熄灭，也不知道烧了多少人家。只可怜隔壁的甄家，早已经烧成了一片瓦砾场。只有他们夫妇和几个家人的性命没有受到伤害。急得甄士隐只能跺脚长叹罢了。只得和妻子商量，暂且到田庄上去安身。偏偏赶上近年水旱灾害不断，收成不好，盗贼蜂拥而起，到处都是抢田夺地、偷鸡摸狗的人，百姓无法安居乐业，因此官兵前来剿捕，他们难以安身。甄士隐只得把田庄全都变卖了，然后带着妻子和两个丫鬟投奔他岳父家去了。
> ```

>[!textus]
>他岳丈名唤封肃，本贯大如州人氏，虽是务农，家中都还殷实。今见女婿这等狼狈而来，心中便有些不乐。幸而士隐还有折变田地的银子未曾用完，拿出来托他随分就价薄置些须房地，为后日衣食之计。那封肃便半哄半赚，些须与他些薄田朽屋。士隐乃读书之人，不惯生理稼穑等事，勉强支持了一二年，越觉穷了下去。封肃每见面时，便说些现成话，且人前人后又怨他们不善过活，只一味好吃懒作等语。士隐知投人不着，心中未免悔恨，再兼上年惊唬，急忿怨痛，已有积伤，暮年之人，贫病交攻，竟渐渐的露出那下世的光景来[^82]。
> ```
> 他的岳父名叫封肃，是大如州人，虽然以务农为生，但家中还算富裕。如今见女婿这般狼狈地前来投奔，心中便有些不高兴。幸亏甄士隐还有变卖田地的银子没用完，拿出来托付给封肃，让他根据行情便宜购置些房产土地，作为日后的衣食之资。那封肃便半哄半骗，只给了他一些薄田和几间破房子。甄士隐是个读书人，不熟悉经营生计、耕种庄稼等事，勉强维持了一两年，越发贫穷了。封肃每次见面时，就说些现成的风凉话，而且在人前背后又埋怨他们不会过日子，只一味地好吃懒做之类的话。甄士隐知道投靠他人没有依靠，心中不免悔恨，再加上去年受到惊吓，急怒、怨恨、痛苦交织，已有积伤，他已是暮年之人，贫病交加，竟渐渐露出了不久于人世的迹象。
> ```

>[!textus]
>可巧这日拄了拐杖挣挫到街前散散心时，忽见那边来了一个跛足道人，疯癫落脱[^83]，麻<ruby>屣<rt>xǐ</rt></ruby><ruby>鹑<rt>chún</rt></ruby>衣[^84]，口内念着几句言词，道是：
>
> 世人都晓神仙好，惟有功名忘不了！
> 古今将相在何方？荒冢一堆草没了。
> 世人都晓神仙好，只有金银忘不了！
> 终朝只恨聚无多，及到多时眼闭了。
> 世人都晓神仙好，只有姣妻忘不了！
> 君生日日说恩情，君死又随人去了。
> 世人都晓神仙好，只有儿孙忘不了！
> 痴心父母古来多，孝顺儿孙谁见了？
> ```
> 可巧这一天，他拄着拐杖挣扎着走到街前散散心，忽然看见那边来了一个瘸腿的道士，疯疯癫癫、无拘无束，穿着麻鞋、衣衫破旧，嘴里念着几句言辞，说道：
> 
> 世人都知道神仙生活逍遥自在，只是忘不了功名利禄。
> 古往今来的将军和宰相如今都在哪里呢？只剩下一堆荒坟被野草掩盖了！
> 世人都知道神仙的日子好，只是对金银财宝却难以忘怀。
> 整天只遗憾聚敛的钱财不够多，等到聚敛得多了却已经闭眼离世了。
> 世人都知道神仙的日子好，只有美貌的妻子忘不了。
> 夫妻每天都诉说着彼此的恩情，可丈夫一死妻子就改嫁他人了。
> 世人都知道神仙的日子好，只有对儿孙的牵挂忘不了。
> 自古以来，痴心疼爱子女的父母有很多，可是孝顺的子孙又有谁见过呢！
> ```

>[!textus]
>士隐听了，便迎上来道：“你满口说些什么？只听见些‘好’‘了’‘好’‘了’。”那道人笑道：“你若果听见‘好’‘了’二字，还算你明白。可知世上万般，好便是了，了便是好。若不了，便不好；若要好，须是了。我这歌儿，便名《好了歌》”士隐本是有宿慧[^85]的，一闻此言，心中早已彻悟[^86]。因笑道：“且住！待我将你这《好了歌》解注出来何如？”道人笑道：“你解，你解。”士隐乃说道：
>
陋室空堂，当年<ruby>笏<rt>hù</rt></ruby>满床[^87]，衰草枯杨，曾为歌舞场。
蛛丝儿结满雕梁，绿纱今又糊在蓬窗上。
说什么脂正浓，粉正香，如何两鬓又成霜？
昨日黄土<ruby>陇<rt>lǒnɡ</rt></ruby>头[^88]送白骨，今宵红灯帐底卧鸳鸯。
金满箱，银满箱，展眼乞丐人皆谤。
正叹他人命不长，那知自己归来丧！训有方，保不定日后作强梁[^89]。
择膏粱[^90]，谁承望流落在烟花巷[^91]！因嫌纱帽小，致使锁枷杠；昨怜破袄寒，今嫌紫蟒[^92]长：乱烘烘你方唱罢我登场，反认他乡是故乡[^93]。甚荒唐，到头来都是为他人作嫁衣裳[^94]！
> ```
> 士隐听了，便迎上去问道：“你嘴里说的是什么？只听见说些‘好’‘了’‘好’‘了’。那道人笑着说：“你如果真听清了‘好’‘了’两个字，还算你明白。”要知道世上的各种事物，好就是了，了就是好。如果不能了结，就不会好；如果想要好，就必须了结。我这首歌就叫《好了歌》。”士隐本来就有悟性，听了这话，心中早就顿悟了。于是笑着说：“且住！”等我来为你的《好了歌》做个解注怎么样？道人笑着说：“你解，你解。”士隐便说道：
> 
> 如今这些破陋的屋子和空荡荡的厅堂，当年可是笏板堆满床，如今枯萎的野草和干枯的杨树所在之处，曾经是歌舞升平的场所。
> 雕花的房梁上已经结满了蜘蛛网，如今又把绿纱糊在那简陋的窗户上。
> 说什么脂粉正浓艳芳香，怎么转眼间两鬓就已经斑白如霜了呢？
> 昨天黄土陇头还堆积着白骨，今晚还在红烛映照的帐子里成双成对地安卧。
> 黄金满箱，白银满箱。转眼间就沦为乞丐遭人毁谤。
> 正在感叹别人寿命不长，哪里知道自己很快也会死去！即使教导有方，也保不定日后会成为强盗。
> 精心挑选富贵人家的子弟嫁，谁能料到最终会流落到妓院之中！因为嫌官职小拼命往上爬，结果反而被戴上枷锁锒铛入狱。昨天还哀叹破旧棉袄难以御寒，今天却嫌弃紫色蟒袍太长。乱哄哄的，你刚唱完，我又登场。反而把他乡认作了故乡。实在是荒唐，到最后都是为别人做了嫁衣啊！
> ```

>[!textus]
>那疯跛道人听了，拍掌笑道：“解得切，解得切！”士隐便说一声“走罢！”将道人肩上褡裢[^95]抢了过来背着，竟不回家，同了疯道人飘飘而去。当下烘动街坊，众人当作一件新闻传说。封氏闻得此信，哭个死去活来，只得与父亲商议，遣人各处访寻，那讨音信？无奈何，少不得依靠着他父母度日。幸而身边还有两个旧日的丫鬟伏侍，主仆三人，日夜作些针线发卖，帮着父亲用度。那封肃虽然日日抱怨，也无可奈何了。
> ```
> 那疯癫的跛足道人听了，拍着手笑道：“解得贴切，解得贴切！”士隐便笑了一声说“走罢！”从道人肩膀上抢过褡裢背在身上，竟不回家，和疯道人一起飘飘然离去了。当下这件事轰动了街坊邻居，大家都把它当作一件新鲜事来传扬。封氏听到这个消息，哭得死去活来，只得和父亲商量，派人到各处去寻访，哪里能得到音信呢？没有办法，封氏只好依靠着她的父母过日子。幸亏身边还有两个旧日的丫鬟伺候，主仆三人，日夜做些针线活拿去卖，帮着父亲维持家用。那封肃虽然天天抱怨，也没有办法了。
> ```

>[!textus]
>这日，那甄家大丫鬟在门前买线，忽听街上喝道之声，众人都说新太爷到任。丫鬟于是隐在门内看时，只见军牢快手[^96]，一对一对的过去，俄而大轿抬着一个乌帽猩袍的官府过去。丫鬟倒发了个怔，自思这官好面善，倒像在那里见过的。于是进入房中，也就丢过不在心上。至晚间，正待歇息之时，忽听一片声打的门响，许多人乱嚷，说：“本府太爷差人来传人问话。”封肃听了，唬得目瞪口呆，不知有何祸事。
>
> ```
> 这一天，甄家的大丫鬟在门前买线，忽然听到街上有人吆喝开道的声音，众人都说新太爷到任了。丫鬟于是躲在门内偷看，只见军牢快手一对一对地走过去，不久，一顶大轿抬着一位头戴乌纱帽、身穿猩红色官袍的官员过去了。丫鬟不禁愣住了，心想这位官员模样很眼熟，好像在哪里见过。于是走进房里，也就把这件事抛到脑后不再放在心上了。到了晚上，正准备休息的时候，忽然听到一阵猛烈的敲门声，许多人乱哄哄地叫嚷，说：‘本府太爷派人来传你去问话。’封肃听了，吓得目瞪口呆，不知道有什么祸事。
> ```
